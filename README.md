# eleijonmarck
My personal repository of knowledge

Inspiration taken from http://blog.otoro.net/

## Prerequisite

``` bash
$ brew install npm
$ npm install -g @vue/cli
$ npm install -g @vue/cli-init
```

## Installation

``` bash
$ vue init webpack-simple eleijonmarck
$ npm install
```

## Development of site

``` bash
$ npm run dev
```
